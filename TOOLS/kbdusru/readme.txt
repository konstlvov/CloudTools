0. Запустить cmd.exe с правами администратора, и последующие шаги делать из-под него

1. Файл kbdusru_undead.dll скопировать в C:\Windows\System32:

  C:\Windows\System32\kbdusru_undead.dll

2. Импортировать в реестр файл kbdusru_undead.reg

3. Открыть контрольную панель (можно с помощью батника open_control_input.bat) и настроить следующим образом:
   * Оставить среди установленных языков только РУССКИЙ
   * Выбрать для него только что установленную клавиатурную раскладку в методах ввода
